实验一 在Github上用Markdown创建技术博客

一、实验目的

1.熟悉Github网站
2.掌握Markdown语法

二、实验任务

1.注册Github账号：zhouxinvi
2.创建Github Pages repository
3.发布第一篇博客



4.学习Markdown
5.利用Hexo生成博客并部署到Github

三、实验步骤

1.本地配置hexo

1）安装dit
2）安装node
3）设置npm镜像为淘宝
    命令：npm install -g cnpm --registry=https://registry.npm.taobao.org
4）安装hexo
    命令：cnpm install hexo-deployer-git --save
5）初始化站点
    命令：hexo init myhexo
          cd myhexo
          cnpm install
6）浏览本地站点
    命令：hexo server

2.在Github上部署hexo

1）在Github上新建repository，名称为zhouxinvi.github,io
2）修改配置文件_config.yml的deploy部分


3)安装hexo-deployer-git
    命令：cnpm install hexo-deployer-git --save
4)配置git全局邮箱与名字
     命令：git config --global user.email “1241455969@qq.com”
           git config --global user.name “Xin Zhou”
5)生成博客
    命令：hexo generate
6)部署博客
    命令：hexo deploy

3.添加新文章

1)生成新文章
     命令：hexo new newblog
2)用编辑器修改文章：myhexo\source_posts\newblog.md
3)生成并发布到Github
     命令：hexo d -g

四、实验小结

总结：学习了markdown的简明语法，搭建了自己的博客。
